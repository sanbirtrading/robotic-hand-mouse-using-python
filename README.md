# Robotic Hand Mouse using python #

Using this script we can simply control the cursor pointer using our hand

### Packages to Install to setup the script ###

To Run the Python Script you'd requires an Python code editor such as PyCharm Editior. And You needs to Install the packages below:

* Opencv Python Package (https://github.com/opencv/opencv-python)
* MediaPipe Package (https://github.com/google/mediapipe)
* PyAutoGui Package (https://github.com/asweigart/pyautogui)

### How do I run the code? ###

To run the code you requires to click on ``Run`` terminal or incase by pressing keyboards ``Shift + F10`` (If it's at PyCharm)
